Detailed Description

Core System Information Component Provides system metrics and hardware information without WebUI dependencies.

Collects ESP32 system metrics (uptime, heap, chip info) for dashboards and logs.

Designed as a lightweight diagnostic component. When paired with a WebUI provider it exposes real-time metrics across dashboard sections and WebSocket updates.
