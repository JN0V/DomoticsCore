Detailed Description

Central runtime for the DomoticsCore framework.

Manages configuration, logging, and registration/lifecycle of components stored in the Components::ComponentRegistry. Provides convenience helpers for accessing components by name or type and drives their begin(), loop(), and shutdown() methods.
