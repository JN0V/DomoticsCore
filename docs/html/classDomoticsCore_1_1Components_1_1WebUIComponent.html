Detailed Description

Async web server + WebSocket frontend that aggregates IWebUIProvider contexts.

Serves embedded HTML/CSS/JS assets, registers component providers, and pushes real-time updates to connected clients. Acts as both a component and a provider to expose global WebUI settings.
