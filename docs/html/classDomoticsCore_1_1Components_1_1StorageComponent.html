Detailed Description

Key-value storage manager wrapping ESP32 Preferences with caching and maintenance.

Opens an ESP32 Preferences namespace, provides typed getters/setters, optional auto-commit, and periodic maintenance/status reporting. Can be paired with a WebUI provider to expose diagnostic data and CRUD helpers.
